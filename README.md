# 📊 Customer Churn Analysis

## 🚀 Project Overview

This project was developed as part of the Future Interns Data Science & Analytics Internship. The objective of this project is to analyze customer churn patterns, identify retention factors, and generate business insights using SQL Server and Power BI.

---

## 🎯 Objectives

- Analyze customer churn patterns.
- Calculate churn and retention rates.
- Study customer demographics.
- Analyze subscription and contract preferences.
- Develop an interactive Power BI dashboard.
- Generate business insights for customer retention.

---

## 🛠️ Tools Used

- SQL Server
- Power BI
- Microsoft Excel
